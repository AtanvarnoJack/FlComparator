Fl Equipement Comparator:

Run: Fl_Equipement_Comparator.jar
Need java jdk 1.8 or jre 1.8

But: comparer des colonnes du FL.

Settings:
Permet de choisir les colonnes du FL à comparer.
	- Save : Sauvegarde la configuration choisie.
	- ReInit Réinitialise la configuration de base.
	- Path Réf : Permet de choisir un FL de référence pour la sélection des champs.
	- Le texte à droite de à Path "Rèf" indique le fichier choisi pour référence.
	- L'arbre de gauche permet de choisir les colonnes a ajoute a la liste de comparaison.
		- Double cliquez sur un label de la liste pour l'ajout à la liste des comparaisons.
	- L'arbre de droite affiche la liste des champs à comparer. (Feuille: champs)
		- Double cliquez sur un label pour le supprimé de la liste.

A la premiére ouverture de "Settings" le programme va vous demander de choisir un FL de Référence. Ex: Thot\\:\ROC\Fichier_Liaison_V4\Fichier de Liaison Standard\Fichier de Liaison V4_15.xlsm

Open Repositories:
 Permet de choisir un répertoire pour lancer la comparaison des clients.
Ce répertoire doit contenir la liste des répertoires client.
(Ex: Thot\\:\ROC\Fichier_Liaison_V4\Clients_ROC).
